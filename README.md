# 🎲 Color Dice Game

Красивая игра с кубиками и выбором цветов. Выбирайте цвета, которыми вы владеете, и при каждом броске кубики не будут показывать эти цвета!

## 🎮 Как играть

1. **Выберите цвета справа** - нажимайте на цвета, которыми вы владеете
2. **Нажмите "ROLL DICE"** - обычный бросок кубиков
3. **Или "MEGA ROLL"** - эффектный бросок с частицами
4. **Кубики никогда не покажут выбранные вами цвета!**

## 🚀 Как запустить

1. Перейдите в Settings вашего репозитория
2. Найдите раздел "GitHub Pages"
3. Выберите Source: "main branch"

## 🔧 Как работает логика

### Выбор цветов
Когда вы нажимаете на цвет, он добавляется в список "владеемых" цветов.

### Бросок кубиков
1. Получается список доступных цветов (все цвета минус выбранные)
2. Кубики крутятся и показывают случайные доступные цвета
3. Результат сохраняется в localStorage

## 💾 Сохранение данных

Ваши выбранные цвета и количество бросков автоматически сохраняются в браузер.

Если вы очистите кэш браузера - данные исчезнут.

## 🎵 Звуки

Игра издает звуки при:
- Нажатии кнопок
- Броске кубиков
- Мега броске
- Выборе/отмене цветов

Звук можно отключить через системные параметры браузера.
